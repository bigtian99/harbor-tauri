# 发布 / 改动冒烟清单

每次合并构建相关改动或发版前勾选。

## 构建与推送

- [ ] **上传推送（JAR）**：选 JAR → 构建镜像 → 推送 Harbor → 历史有记录
- [ ] **上传推送（前端 dist）**：选 dist 目录 → nginx 镜像推送成功
- [ ] **镜像推送**：本地镜像列表 → tag 推送 Harbor
- [ ] **分支打包（Maven）**：指定分支 → `mvn` 产物输出 → 可选联动推送
- [ ] **分支打包（npm）**：install/缓存命中 → `npm run` 产物
- [ ] **分支打包（npm+后端）**：前后端并行产物均存在

## 运营

- [ ] **落地页生成**：渠道 ID → 模板渲染 → iframe 预览正常
- [ ] **落地页 FTP**：上传成功（配置/环境可用时）
- [ ] **结算单**：双 Excel → 输出目录生成文件
- [ ] **打包加速**：批量任务进度与结果

## 其它

- [ ] **历史记录**：列表、删除单条/清空
- [ ] **设置**：Harbor 配置保存/重开仍在
- [ ] **系统日志**：有 `[build]` / `[landing]` 等模块日志；无明文 password
- [ ] **检查更新**（有新版本环境时）

## OPS 构建

- [ ] `OPS_MODE` 包仅显示运营相关菜单（对照 [ops-vs-full.md](./ops-vs-full.md)）
